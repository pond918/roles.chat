chat-roles-repo

restAPI服务，使用和创建社区中，每个人共享的rolebots机器人

chat-roles-repo提供了一组restAPI服务, 让每一个人都能方便快捷地使用社区共享的chat-roles rolebots。

chat-roles 有什么用

我们都知道，当我们给LLMs（如chat-GPT），提一个问题，它能帮我们回答问题或者生成文章。

那么如果我们向一个chat-role提一个要求，这个rolebot自动将这个要求转化为成百上千个小任务，交给其他的rolebots共同完成，rolebots各司其职，像一个团队一样合作。

最后交付给你的结果，一定是前所未有的。

chat-roles-repo 的优势是什么

基于chat-roles-repo服务，我们定义了roles, hosts, actors结构，目的是让社区中每一个人，都能方便地共享和使用强大的，目标明确的rolebots。

用流行的话来说，就是

Prompt as Code

人人都能编写强大的智能机器人。
